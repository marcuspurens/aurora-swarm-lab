# Aurora minnesfraser (snabbcopy)

Anvand exakt detta monster:

`kom ihag detta: <regel eller preferens>`

## 10 klara fraser

1. `kom ihag detta: jag foredrar korta svar pa svenska.`
2. `kom ihag detta: nar jag skriver "pipeline-check", kor transkribering och kontrollera hela pipeline.`
3. `kom ihag detta: visa alltid filvag till output i slutet.`
4. `kom ihag detta: forklara utan utvecklarslang nar det gar.`
5. `kom ihag detta: prioritera ORF/EBUCore+-case i denna session.`
6. `kom ihag detta: om nagot misslyckas, ge mig exakt kommando for att testa igen.`
7. `kom ihag detta: jag vill se entities, relations och ontology compliance i rapporten.`
8. `kom ihag detta: sammanfatta transkribering i markdown med rubriker och punktlista.`
9. `kom ihag detta: jag vill alltid veta var artefakter sparas pa disk.`
10. `kom ihag detta: fraga mig innan du andrar struktur i vaulten.`

## Nar du vill bli pamind

Skriv:

`paminn mig om minnesfrasen`

eller:

`visa minnesfraser`

## Snabbformat for input till Aurora

Klistra in:

`input: <url eller filvag>`
`tags: <tag1, tag2>`
`context: <kort beskrivning>`
`mal: <vad du vill fa ut>`
